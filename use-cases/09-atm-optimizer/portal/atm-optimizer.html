🔧 MCP Tools

execute_sql_query

Run read-only SQL against TradeFinance DB. Blocks: DROP, DELETE, INSERT, UPDATE, ALTER, TRUNCATE.

get_schema_info

Discover tables and columns. No args → all tables. With table_name → column details.

analyze_blob_data

Extract VARBINARY content (PDF documents). Returns content_type, size, and text preview.
